pdf2docx.layout.Column module

Column of Section.

In most cases, one section per page. But in case multi-columns page, sections are used to distinguish these different layouts.

Note

Currently, support at most two columns.

{
    'bbox': (x0, y0, x1, y1),
    'blocks': [{
        ... # block instances
    }, ...],
    'shapes': [{
        ... # shape instances
    }, ...]
}
