pdf2docx.text.Line module¶

Text Line objects based on PDF raw dict extracted with PyMuPDF. @created: 2020-07-22

—

Data structure of line in text block: {

‘bbox’: (x0,y0,x1,y1), ‘wmode’: m, ‘dir’: [x,y], ‘spans’: [ spans ]

}

https://pymupdf.readthedocs.io/en/latest/textpage.html
