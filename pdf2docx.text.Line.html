pdf2docx.text.Line module¶

Text Line objects based on PDF raw dict extracted with PyMuPDF.

Data structure of line in text block referring to this link:

{
    'bbox': (x0,y0,x1,y1),
    'wmode': m,
    'dir': [x,y],
    'spans': [ spans ]
}
