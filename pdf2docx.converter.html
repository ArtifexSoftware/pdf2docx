pdf2docx.converter module¶

class pdf2docx.converter.Converter(pdf_file: str)¶

Bases: object

Read PDF file pdf_file with PyMuPDF to get raw layout data page by page, including text, image and the associated properties, e.g. bounding box, font, size, image width, height, then parse it with consideration for docx re-generation structure. Finally, generate docx with python-docx.
