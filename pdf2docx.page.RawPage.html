pdf2docx.page.RawPage module¶

A wrapper of fitz.Page to extract source contents.

The raw page content extracted with PyMuPDF API page.getText('rawdict') is described per link https://pymupdf.readthedocs.io/en/latest/textpage.html:

{
    # raw dict
    ----------------------------
    "width" : w,
    "height": h,
    "blocks": [{...}, {...}, ...],

    # introduced dict
    ----------------------------
    "shapes" : [{...}, {...}, ...]
}

In addition to the raw layout dict, rectangle shapes are also included.
