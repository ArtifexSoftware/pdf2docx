Extract table
======================

::

    from pdf2docx import Converter

    pdf_file = '/path/to/sample.pdf'

    cv = Converter(pdf_file)
    tables = cv.extract_tables(start=0, end=1)
    cv.close()

    for table in tables:
        print(table)
