pdf2docx.page.Page module¶

Page object parsed with PDF raw dict.

In addition to base structure described in RawPage, some new features are also included, e.g.

  • page margin

  • parsed table block and nested layout

{
    # raw dict
    ----------------------------
    "width" : w,
    "height": h,
    "blocks": [{...}, {...}, ...],

    # introduced dict
    ----------------------------
    "id": 0, # page index
    "margin": [left, right, top, bottom],
    "shapes" : [{...}, {...}, ...]
}
