pdf2docx.image.Image module¶

Extract images from PDF and define the Image object.

Image properties could be extracted with PyMuPDF method page.getText(‘rawdict’):

``` # data structure defined in link: # https://pymupdf.readthedocs.io/en/latest/textpage.html

{

‘type’: 1, ‘bbox’: (x0,y0,x1,y1), ‘ext’: ‘png’, ‘width’: w, ‘height’: h, ‘image’: b’’,

# — discard properties — ‘colorspace’: n, ‘xref’: xref, ‘yref’: yref, ‘bpc’: bpc

}¶

But, the extracted image bytes may be different from the source images in PDF, especially png images with alpha channel. So, images are extracted separately, and stored in a similar structure.
