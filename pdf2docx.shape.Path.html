pdf2docx.shape.Path module¶

Objects representing PDF path (both stroke and filling) extracted from both pdf drawings and annotations.

@created: 2020-09-22

—

Data structure based on results of page.getDrawings(): {

‘color’: (x,x,x) or None, # stroke color ‘fill’ : (x,x,x) or None, # fill color ‘width’: float, # line width ‘closePath’: bool, # whether to connect last and first point ‘rect’ : rect, # page area covered by this path ‘items’: [ # list of draw commands: lines, rectangle or curves.

(“l”, p1, p2), # a line from p1 to p2 (“c”, p1, p2, p3, p4), # cubic Bézier curve from p1 to p4, p2 and p3 are the control points (“re”, rect), # a rect

}
