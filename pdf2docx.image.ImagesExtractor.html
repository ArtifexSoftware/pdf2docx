pdf2docx.image.ImagesExtractor module¶

Extract images from PDF.

Both raster images and vector graphics are considered:

  • Normal images like jpeg or png could be extracted with method page.getText('rawdict') and Page.getImageList(). Note the process for png images with alpha channel.

  • Vector graphics are actually composed of a group of paths, represented by operators like re, m, l and c. They’re detected by finding the contours with opencv.

Extract normal images with Page.getImageList().

Note

extract_vector_graphics(exclude_areas: list, clip_image_res_ratio: float = 3.0)¶

Detect and extract vector graphics by clipping associated page area.

Parameters

  • exclude_areas (list) – A list of bbox-like (x0, y0, x1, y1) area to exclude, e.g. raster image area, table area.

  • clip_image_res_ratio (float, optional) – Resolution ratio of clipped bitmap. Defaults to 3.0.

Returns

A list of extracted and recovered image raw dict.

Return type

list

Note

Contours for vector graphics are detected first with opencv-python.
